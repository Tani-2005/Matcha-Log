# 🍵 Ghibli Matcha Log

A cozy, Studio-Ghibli–inspired web app where you can log your favourite matcha tins, rate them, and view them in a beautiful animated environment.

This project combines **clean JavaScript logic** with **aesthetic UI design**, including floating sakura petals, animated steam, and a calming matcha café vibe.

---

## ✨ Features

### 🍵 Matcha Tracker

* Add matcha tins with:

  * Name
  * Brand
  * Rating (1–5 stars)
* View all entries in a clean animated list
* Delete entries anytime
* Automatically saved using **LocalStorage**

### 🔎 Sorting Options

* Newest first
* Rating (High → Low)
* Rating (Low → High)
* Name (A → Z)

### 🌸 Ghibli-Inspired Animations

* Floating tea cup animation
* Moving curved steam
* Animated tea ripple
* Random falling sakura petals
* Glassmorphism UI style
* Smooth hover interactions

## 🚀 How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`

That’s it — no setup or installation needed.

* No database needed
